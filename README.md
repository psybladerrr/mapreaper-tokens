# MapReaper Token Storage

Приватный репозиторий для хранения токенов активации расширения MapReaper.

## Структура

- `tokens/active.json` - активные токены
- `tokens/users.json` - данные пользователей
- `tokens/revoked.json` - отозванные токены

## Безопасность

⚠️ **ВАЖНО**: Этот репозиторий должен оставаться приватным!

Не делитесь содержимым файлов токенов с третьими лицами.
